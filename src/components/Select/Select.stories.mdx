# Select

The `Select` component allows users to pick a value from a set of predefined options.

## Props

<ArgsTable of={Select} />

Where `Options` is:

```ts
interface Options {
  value: string
  text: string
}
```

In addition, any property of the [Chakra UI Select component](https://chakra-ui.com/docs/form/select#props) can be used.

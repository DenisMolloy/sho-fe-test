# Icon

Icon component is used to display icons. Behind the scenes it uses `@chakra-ui/icons` library.

## Usage

Specify the icon name as prop to render specific icon.

## Props

<ArgsTable of={Icon} />

In addition, any property of the [Chakra UI Icon component](https://chakra-ui.com/docs/media-and-icons/icon#using-chakra-ui-icons) can be used to customize size, colors and other styles.

## Examples

### Fallback icon

Without any `props` it renders a fallback icon.

<Canvas>
  <Story name="Fallback icon">
    <Icon />
  </Story>
</Canvas>

### Default style

Icon with default styles.

<Canvas>
  <Story name="Default style" args={{ icon: 'CheckCircleIcon' }} decorators={[]}>
    {args => <Icon {...args}></Icon>}
  </Story>
</Canvas>

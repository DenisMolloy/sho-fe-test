# Heading

Headings are used for rendering headlines. It renders an `h2` tag by default.

## Props

<ArgsTable of={Heading} />

In addition, any [Chakra UI Heading component](https://chakra-ui.com/docs/typography/heading#props) property and all attributes supported by the [Heading elements](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/Heading_Elements#attributes) can be used.

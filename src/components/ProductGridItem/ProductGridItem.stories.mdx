# ProductGridItem

The `ProductGridItem` is used internally in the `ProductGrid` component and it renders a product card.

## Usage

`ProductGridItem` accepts a single `product` to display. See a product example on the <LinkTo kind="docs-models--page#product">**_Models page_**</LinkTo>.

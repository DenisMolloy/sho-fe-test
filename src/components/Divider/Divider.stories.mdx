# Divider

This component is useful for separating content. It renders by default a `hr` tag.

## Usage

It can display a thin horizontal or vertical line.

## Props

<ArgsTable of={Divider} />

In addition, any property of the [Chakra UI Divider component](https://chakra-ui.com/docs/data-display/divider) can be used.

# Grid

A primitive useful for grid layouts. Grid is Container with display: grid and it comes with helpful style shorthand. It renders a div element.

## Props

<ArgsTable of={Grid} />

In addition, any property of the [Chakra UI Grid component](https://chakra-ui.com/docs/layout/grid#props) can be used.

## Examples

You can use `Grid` with any child components, here you can see an example using the `Container` component.

<Canvas>
  <Story
    name="with Container"
    argTypes={{
      w: {
        control: 'text',
      },
      bg: {
        control: 'text',
      },
    }}
    args={{
      templateColumns: 'repeat(5, 1fr)',
      gap: '6',
      h: '10',
      w: '100%',
      bg: 'blue.500',
    }}
  >
    {args => (
      <Grid templateColumns={args.templateColumns} gap={args.gap} h={args.h}>
        <Container w={args.w} bg={args.bg} />
        <Container w={args.w} bg={args.bg} />
        <Container w={args.w} bg={args.bg} />
        <Container w={args.w} bg={args.bg} />
        <Container w={args.w} bg={args.bg} />
      </Grid>
    )}
  </Story>
</Canvas>

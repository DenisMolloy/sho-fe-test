# BlogPostPreview

The `BlogPostPreview` Component displays a preview card for a blog post on /blog and requires data from a `Blog` content object.

## Props

<ArgsTable of={BlogPostPreview} />

`BlogPostPreview` expects name, title, sport, author, authorImage, createdDate, summary, and heroImage props.

# NumberInput

**NumberInput** component is an enhanced Input component which allows changing the value by clicking "-" and "+" buttons.

## Props

<ArgsTable of={NumberInput} />

In addition, any [Chakra UI NumberInput component](https://chakra-ui.com/docs/form/number-input) property can be used.

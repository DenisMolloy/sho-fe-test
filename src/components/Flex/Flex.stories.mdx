# Flex

Flex is Container with `display: flex` and comes with helpful style shorthand. It renders a `div` element.

## Props

<ArgsTable of={Flex} />

In addition, any property of the [Chakra UI Flex component](https://chakra-ui.com/docs/layout/flex#props) can be used.

## Examples

<Canvas>
  <Story
    name="default"
    argTypes={{
      justify: {
        control: {
          type: 'select',
          options: ['start', 'center', 'space-around', 'space-between', 'space-evenly'],
        },

        control: {

    {args => (
      <Flex {...args}>

        </Container>

          Container 2
        </Container>
      </Flex>
    )}
  </Story>
</Canvas>

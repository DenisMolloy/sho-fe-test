# Menu

This component displays an accessible dropdown menu that can be used to organize a set of website's navigation links.

## Props

<ArgsTable of={Menu} />

In addition, any property of the [Chakra UI Menu component](https://chakra-ui.com/docs/data-display/Menu) can be used.

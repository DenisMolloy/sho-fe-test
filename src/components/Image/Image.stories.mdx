# Image

The Image component is used to display images. To work properly, the image must be within one of our CDN's valid url:

- assets.frontend.shogun.dev
- frontend.shgcdn.com
- ucarecdn.com
- f.shgcdn.com

## Props

<ArgsTable of={Image} />

Internally Image renders [Chakra's `Img` component](https://chakra-ui.com/docs/media-and-icons/image#usage-with-ssr). Image accepts properties of the [native `img` element](https://html.spec.whatwg.org/multipage/embedded-content.html#the-img-element) and Chakra's `Box` properties.

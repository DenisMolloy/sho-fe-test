# Carousel

Carousel allows multiple images to be shown in a single coveted space. It has the main area where a big image is shown. Below the main area, it has the slider area where smaller images are shown. The slider area allows navigating through images.

## Usage

The most common use case is to display product images on a Product page.

## Props

<ArgsTable of={Carousel} />

Where MediaItem is

```ts
interface MediaItem {
  name: string
  src: string
  alt?: string
  width: number
  height: number
}
```

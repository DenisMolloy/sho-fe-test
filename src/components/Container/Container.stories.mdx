# Container

Container is the most abstract component on top of which many other components are built.
It renders a `div` element by default.

Behind the scenes it uses [Box](https://chakra-ui.com/docs/layout/box) or [Container](https://chakra-ui.com/docs/layout/container) components from **Chakra-UI** depending on the passed props.

## Usage

Container is useful in the following common cases:

- Create visual block with/without background/border
- Create responsive layouts
- Create a visual block with constrained content
- Provide a shorthand way to pass styles via props (`bg` instead of `backgroundColor`).
- Compose new component and allow for override using the `as` prop

# Link

Links are accessible elements used primarily for navigation and renders an `a` tag.

## Props

<ArgsTable of={Link} />

In addition, all attributes supported by the [`a` element](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a#attributes) can be used.

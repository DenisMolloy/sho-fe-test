# ProductBox

The `ProductBox` section is a composable component that shows a detailed view of a product and allows the user to buy the product by clicking on the `Add to Cart` button.

## Usage

`ProductBox` accepts a `product` prop which is an object holding product data. See a product example on the <LinkTo kind="docs-models--page#product">**Models page**</LinkTo>.
